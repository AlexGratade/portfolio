Description du projet

Application Symfony pour la gestion d’un jardinier indépendant : clients, prestations, factures, règlements.

🛠️ Technologies utilisées

  • Symfony 6
  • Doctrine ORM
  • MySQL
  • HTML/CSS

✨ Fonctionnalités principales

  • CRUD clients
  • Gestion devis/factures
  • Suivi des règlements
  • Interface admin sécurisée
